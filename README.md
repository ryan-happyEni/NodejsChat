# NodejsChat
Chat-Server with Node.js
Node.js 채팅 서버입니다.

## Install the three modules :
    - npm install express
    - npm install request
    - npm install socket.io
    - npm install cheerio

## How to use :
    - npm start
    
## Functions
    - 기본적인 채팅 기능, 특정 방(방아이디) 구성원끼리 메세지 주고받기
    - 메세지에 URL 포함시 미리보기 기능
